### **Project Summary**

Event Handling Web App is a modern, fully responsive event management and service showcase platform built with Next.js. The application is designed for event management companies, wedding planners, and service providers to showcase their services, manage event portfolios, and engage with clients through an elegant, interactive interface. It features smooth animations, dark/light mode support, and intuitive navigation across multiple service categories.

---

### **Tech Stack**

**Frontend Framework:**
- Next.js 14.2.25 (with App Router)
- React 19 (latest version)
- TypeScript (type-safe development)

**UI & Styling:**
- Tailwind CSS 4.1.9 (utility-first CSS framework)
- Tailwind CSS Animate (smooth animations)
- Radix UI (accessible component primitives)
- Lucide React (modern icon library)
- Embla Carousel React (carousel functionality)

**Backend**
- Node.js
